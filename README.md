# Image-Manager
"Görsel Yönetici (Visual Manager) is a comprehensive visual asset management tool designed for designers and developers. It aims to keep complex image files organized, facilitate quick classification, and provide easy access to project files. Developed under the assurance of Pixel Neon for a more efficient design workflow."

Key Features
Based on the functionality visible in image_0eef16.png, the application includes the following features:

Advanced Scanning: Easily scan your entire computer or specific drives to locate all image assets.

Smart Selection Tools: Select all files, clear selections, or choose specific files with intuitive one-click buttons.

Duplicate Detection: Automatically identify and manage duplicate image files to save storage and keep projects clean.

Batch Operations: Quickly delete or copy selected images, streamlining your asset management tasks.

Paged View: Efficiently browse through large image collections with the built-in pagination system (e.g., viewing 1214 images across multiple pages).

Path Tracking: Keep track of the source directories for your files, ensuring you always know where your assets are located.

Theme Support: Customize your experience with different interface themes (e.g., Dark Mode).

Technical Overview
Purpose: Asset management and workflow optimization.

Ecosystem: Part of the Pixel Neon Entertainment toolset.

Focus: Speed, reliability, and ease of use for designers and developers.

How to Use
Scan: Click "Tüm Bilgisayarı Tara" or "Sürücüleri Seç" to populate your library.

Browse: Use the pagination controls at the top to navigate through your assets.

Manage: Select desired files using the checkboxes or selection tools, then perform actions like "Seçilenleri Sil" (Delete) or "Seçilenleri Kopyala" (Copy).

Cleanup: Use the "Yinelenenler" feature to find and remove redundant files.
